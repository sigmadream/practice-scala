# 3장 연습문제 

## Q1

```scala
val input = ""

val result = input match {
    case "" => "n/a"
    case name => name
}

print(result)
```

## Q2

```scala
val amount = 2.143
val result1 = if (amount > 0) "greater" else if (amount < 0) "lesser" else "same"
val result2 = amount match {
    case x if (x > 0) => "greater"
    case x if (x < 0) => "lesser"
    case x => "same"
}

print(result1, result2)
```

## Q3

```scala
val color = "magenta"

val result = color match {
    case "cyan" => "00ffff"
    case "magenta" => "00ff00"
    case "yellow" => "ffff00"
    case x => s"Didn't expect $x !"   
}
print(result)
```

## Q4

```scala
for (i <- 1 to 100 by 5) {
    for (j <- i to (i + 4)) {
      print(s"$j, ")
    }
    println("")
}
```

## Q5

```scala
for (i <- 1 to 100) {
    i match {
      case x if x % 15 == 0 => println("typesafe")
      case x if x % 5 == 0 => println("safe")
      case x if x % 3 == 0 => println("type")
      case x => println(x)
    }
}
```
